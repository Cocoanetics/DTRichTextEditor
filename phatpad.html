PhatPad Overview

PhatPad is an advanced note-taking, presentation, and collaboration software for iPad that allows you to create content-rich documents using handwritten and printed text, drawings, images, and annotations. PhatPad also supports the Presentation mode, which allows you to create quick presentations and display them on the iPad screen or an external VGA monitor. In addition, the included handwriting recognition engine allows you to convert previously recorded handwritten notes into digital text, and edit text using the handwriting input panel in addition to a keyboard. PhatPad documents can be synchronized with the local computer via iTunes (or with the Dropbox cloud), exchanged between 2 or more devices on the local WiFi network, printed, emailed, or exported to PDF.

Benefits and Features

  • Create, edit, print, and share content-rich documents using handwritten and printed text, drawings, images, annotations, and more.
  • Take advantage of advanced document editing features including multilevel undo/redo.
  • Draw perfect geometrical shapes utilizing the shape recognition feature.
  • Organize documents in the hierarchical folder structure and search documents for text.
  • Use advanced handwriting recognition to convert previously recorded handwritten notes to digital text.
  • Share your documents with others in PDF format.
  • Synchronize your documents with the local computer via iTunes, or with the Dropbox cloud.
  • Make presentations and use the presentation mode to show them on the iPad screen or external VGA monitor.

Using PhatPad

PhatPad has two main operating modes: pen mode, and drag/select mode. The Pen mode should be used when handwriting or drawing, the other mode for editing ink, or working with pictures and text labels.

Note: pictures are always drawn below the text labels, and text labels below the ink. If there is more than one picture or more than one text label on the same page, you can change the Z order of overlapping images or text labels but not of overlapping strokes.

In Pen Mode

The Pen mode allows you to write or draw anywhere on the page. All touches are interpreted as pen strokes. You can also use tap and hold gesture to bring up a menu and edit objects on the page.

  • To cut, copy, or delete individual strokes, touch and hold the desired stroke until it is selected and menu appears, then choose the appropriate command from the menu.
  • To move a stroke to a new location, touch and hold until the stroke is selected, then move it to a desired location.
  • To copy, delete, or edit text, touch and hold the desired text object until the menu appears then select the appropriate command.
  • To resize or delete an image, touch and hold the desired image, then select Delete from the menu to delete the image, or use pinch gesture to resize it.
    Note: if the Palm Rest option is ON, the pinch gesture cannot be used to resize images. You can still resize images in the Selection mode (see below).
  • To move text or image to a new location, touch and hold until the object is selected, then drag it to the desired location.
  • To change pen color or width, touch the button in the toolbar. PhatPad remembers the last 50 pen styles used. Touch More in the Pen Color and Width pop up to access previously used styles.
  • To draw perfect geometrical shapes using the shape recognition engine, touch the button in the toolbar then turn the Detect Shapes switch to ON. PhatPad recognizes the following shapes: line, circle, square, triangle, diamond, and arrow. All shapes must be drawn in a single stroke.

In Drag/Select mode

In this mode you can select, move, delete, and resize multiple strokes, text labels, and images. Note that while working with images and text labels it is recommended to use this mode.

  • To select an individual stroke, touch the desired stroke. The selection frame will appear. You can move the selected stroke to a new location, or hold for the menu.
  • To select a text label or an image, tap on the desired object. The selection frame will appear. You can move the selected object or resize it by touching one of the selection frame corners. Hold, to display the menu.
  • To edit the text label, double tap the desired object.
  • To select multiple strokes, touch the empty portion of the page, then drag your finger across strokes to select them. You can resize selected strokes by touching one of the selection frame corners. Hold, to display the menu.
  • To copy or delete a selection, tap and hold for a menu like in the Pen mode.
  • To convert handwritten text to the digital text, select all the strokes you want to convert and choose the Convert to Text menu command. Use the PhatPad Options to adjust handwriting recognition settings.

In Eraser mode

  • Simply draw line thorugh the steokes you want to delete. When you release your finger all crossed steokes will be deleted.

In either Pen or Drag/Select mode

  • To insert a text label, touch the button then select the Insert Text command. The text label will be inserted at the default location in the text edit mode. When the text label is in the edit mode, it is drawn above all other objects, and its background is opaque.
  • To undo the last editing action, touch the Undo button in the toolbar or shake the device.
  • To edit a text label, touch and hold until the menu appears, then select the Edit command.
  • To record a voice note, touch the Microphone button in the toolbar. The Voice Control is displayed while the voice note is being recorded.
    After voice note is recorded, the Microphone button in the toolbar is replaced with the Speaker button. Touch this button to play the voice note.
  • To switch between keyboard and handwriting input panel while editing a text label, touch , then turn the Use Keyboard switch ON or OFF.
  • To change the Style of a text label, touch and hold for the menu, then select the Style command. Using the Styles popup, you can select one of the pre-defined styles or create your own.
  • To insert a picture, touch the button in the toolbar then choose the Insert Picture command.
  • To delete a picture, touch and hold for the menu, then choose the Delete command.
  • To access the Grid Settings touch the Tools->Grid Settings command. You can enable/disable vertical and horizontal gridlines, change distance between gridlines, and turn the Snap to Grid option ON or OFF. Grid settings are stored with the PhatPad document, and can be different in each document.
  • To manage document pages, touch the button in the toolbar, and then choose the Manage Pages command. A new menu will appear, allowing you to add, duplicate, delete or re-arrange the document pages.
  • To add a new page to the end of the document, touch the button while viewing the last document page. A new empty page will appear.
  • To change the page color, tap and hold at the empty section of the page, then select the Page Color menu. This menu also allows you to selectively delete ink, text, or pictures (as applicable to the current page) or paste a previously copied text label or ink.
  • To print the current document, tap the button in the toolbar, and then select the Print Document command. Hold the device horizontally to print in landscape mode, or vertically to print in portrait mode.
  • To send the current document via email, tap the button in the toolbar, and then select the Share via Email command. By default, PhatPad documents are emailed in the PDF format. If you want to email them in the native PhatPad format, turn OFF the Email as PDF switch in the Options/Email Settings view.
  • To manage your documents, touch the button in the toolbar. To open a document, select the desired document from the list. To rename or delete a document, touch the Edit button. To synchronize with Dropbox, touch the Synchronize with Dropbox button. Documents are stored automatically, so no special action is required.
  • To use the Presentation mode, touch the button, and then select the Presentation Mode command. Using the manual Presentation mode:


    • Tap once or swipe left to go to the next page
    • Swipe right to go to the previous page
    • Touch and hold to go to the first page
    • Tap at the top of the screen to display the toolbar
    • To record a new presentation by specifying how long to display each page of the document, touch the Record button the the Toolbar.
    • To specify page timeouts manually, touch the Timeouts button in the toolbar.
    • To play presentation, touch the Play button in the toolbar.

  • Touch done in the toolbar to exit the Presentation mode.
  • To change PhatPad settings, touch the button in the toolbar, then select the PhatPad Options command.

Using handwriting panel to edit text

  • To enter text, write on the input panel between the lines in your preferred handwriting style. You can write one or more words per recognition session, but always complete the entire word in each session and ignore intermediate results because the dictionary is used to find the best results. When you are done writing, touch the button to insert the recognized text at the cursor location. If you always print, you may want to enable the Separate Letters option.
  • If the Input Panel contains handwriting, the button inserts the recognized text at the current cursor location, otherwise it inserts the return character. You can also use the gesture.
  • If the Input Panel contains handwriting, the button deletes the last stroke, otherwise it deletes the character to the left of the cursor. You can also use the long gesture.
  • If the Input Panel contains handwriting, the button clears the input panel and cancels the recognition session, otherwise it inserts the space character.
  • The button inserts the period at the current cursor location. Holding this button displays the punctuation keypad.
  • If the recognizer returned an unexpected result, touch the incorrectly recognized word in the recognition results window to view alternatives.
  • To use the built-in calculator, write something like 23.4+11.3= . The = symbol at the end of the equation is required to invoke the calculator.

Find and Replace

You can search for characters or words in the current document by using the Tools->Find/Replace command. You can search text labels as well as handwritten notes.

To search handwritten text, turn the Search Handwriting switch ON. Note that when searching handwriting, the search is always case-insensitive and the Match Case option is disabled. Also, space characters should not be used when searching for handwriting, you can search for words or partial words.

To replace characters or words in the text labels, switch to the Replace view. Use the Replace button to replace the next occurrence of the specified text, or Replace All to replace all occurrence.

Grid Settings

To access the Grid Settings touch the Tools->Grid Settings command.

  • To show/hide vertical and/or horizontal gridlines use Vertial Grid and Horizonal Grid switches respectively.
  • To change the distance between vertical and/or horizontal gridlines use the V. Spacing and H. Spacing respectively.
  • Turn the Stanp to Grid switch ON if you want objects to snap to gidlines when dragging/resizing.

Grid settings are stored with the PhatPad document, and can be different in each document.

PhatPad Options

Document Sharing

  • Share My Documents - if ON, the documents on your iPad can be accessed from another iPad, or a desktop computer on the local network. Note that this option is disabled if iPad is not connected to the local WiFi network.
  • Keep Awake - turn ON to prevent device from entering the idle mode while sharing your documents. Note that if the device is not connected to an external power source, make sure not to leave the PhatPad application running on your iPad. If the Stay Awake switch is ON for a long time, the device will not hibernate and your battery will be drained quickly.

Document Settings

  • Sort Order - this option allows you to select the document sort order to be used in the My Documents view. Note that this sort order does not apply when you view documents located on the remote device.
  • Show All Folders - if this switch is ON, all folders appear in the My Documents view allowing users to use drag and drop to re-arrange documents and folders in the Edit mode. Otherwise only subfolders of the current folder are visible leaving more space for documents list.

    • General Settings

  • Use the Keyboard to edit text - if this switch is ON, the standard iPad keyboard is used when editing the text label; otherwise, the handwriting recognition panel is used.
  • Hide Page Footer - if this switch is ON, the page footer containing the page number and the document name does not appear at the bottom of each page. This option also applies when printing or exporting to PDF.
  • Erase Recognized Ink - if this switch is ON, all strokes converted to digital text are erased.
  • Ignore single point strokes - if this switch is ON, single point taps on the screen are not recorded.
  • Palm Rest - if this switch is ON, you may be able to rest your palm on the screen while writing. This option is recommended for handwriting notes only, and may not be suitable for drawing or editing. This option is applicable only to Pen mode.
  • Theme Color - allows you to select the desired theme color for the application. Default is Blue.

    Handwriting Recognition Settings

    Language

    • Allows you to select your handwriting recognition language. Your primary language is free, support for additional languages can be purchased $2.99 each. When adding a new language, you may want to reimport words into the user dictionary from iPad's contacts using the Recognizer Data dialog.

    Recognizer Settings

    • Auto Learner - if this switch is ON, PhatNotes will learn your handwriting patterns in an attempt to improve the quality of recognition. It is recommended to use the Recognition Alternatives and the Spell Checker windows to correct wrong results in the beginning to accelerate the learning process.
    • Autocorrector - Use this option to enable the Autocorrector feature. Auto corrector automatically fixes common spelling and recognizer errors.
    • Add Space - if this switch is ON, a space character is added at the end of the recognized text if 2 or more characters have been recognized, (a space is never added after a single character).
    • Separate Letters - use this option if you print instead of using cursive and do not connect individual letters. If this option is ON, connected letters will not be recognized properly.

    Dictionary Settings

    • Only Known Words - if this switch is ON, the recognition output will be limited only to the dictionary words. This improves the quality of the recognition but makes it impossible to write non-dictionary words.
    • User Dictionary - if this switch is ON, the user dictionary is enabled. To edit the user dictionary, use the Dictionary view.
  • Autoinsert Results - if this switch is ON, you can write continuously without using the Return button or gesture to insert recognition results by starting to write a new word on the left side of the blue marker. To move the marker touch and hold it for one second then move horizontally to the desired location.
  • Play Sound - if this switch is ON, devices plays sound in case of a recognition error.

    • Manage User Data

    Use the User Data view to export (import) the User Dictionary and the Autocorrector word list to/from a text file.

    • To share user data, such as User Dictionary, Autocorrector word list, Shorthand list, and learned recognizer statistics, turn the Share User Data switch ON.
    • The user dictionary file format is a simple ASCII text file containing a single word per line, with no spaces.
    • The Autocorrector file format is CSV (comma separated values). Each row contains a pair of words and optional flags (see Autocorrector Word List Editor section).
    • You can also reset the User Dictionary, Autocorrector, and Recognizer learner to default configuration.

    User Dictionary

    The handwriting recognition engine uses a user vocabulary to increase the recognition quality. Forcing the engine to recognize only words in the included dictionary (this version includes a 100,000+ words US English dictionary) improves overall recognition quality, but makes it impossible to write words that are not in the dictionary. Even if you do not enable the Only Known Words option it is recommended that you add to the user dictionary any new words that you may be using often, such as rare last and first names, company and product names, abbreviations, etc.

    To access the User Dictionary Editor, touch the Edit User Dictionary button in the Options view.

    To add new words to the user dictionary, touch the <enter new word> box and type the word you want to add then touch the button on the left.

    Autocorrector Word List Editor

    To access the Autocorrector Word List Editor, touch the Edit Word List button in the Options view.

    While using the Editor:

    • To modify an existing word pair, touch the desired row while in the view mode.
    • To add a new word pair to the word list, or to delete a word pair, touch the Edit button in the title bar.

    When adding or modifying a word pair, the following options can be used.

    • If the Always Replace switch is ON, the word is always replaced; otherwise the word is replaced only if the recognition engine list of alternatives contain the replacement word.
    • If the Ignore Case switch is ON, words are compared in lower case.
    • If the Disabled switch is ON, the word replacement is disabled. This is useful to disable a specific word replacement without removing it from the list.

    • Default Email Addresses - specify default TO, CC, and BCC email addresses.

    Synchronization Settings

    • Enable sync with Dropbox - turn ON to enable synchronization with Dropbox (if disabled, the Synchronize button does not appear in the My Documents view).
    • Dropbox Login Information - enter your Dropbox Email Address and Password. If this information is not specified, the Dropbox Login window will appear each time you synchronize.
    • Remember Password - if this switch is ON, the Dropbox login information will be retained between application sessions.
    • Autosync on Startup - turn ON to automatically synchronize your PhatPad documents every time you start the application. This option is disabled if the Remember Password switch is OFF.
    • Create New Dropbox Account - if you do not already have a Dropbox account, touch this button to create a new one. It is currently free for 2GB of storage. For more information, visit www.dropbox.com.
